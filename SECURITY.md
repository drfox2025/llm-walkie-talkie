# Security Policy

We take security seriously. Please review this document to report vulnerabilities or see currently supported versions.

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | :white_check_mark: Yes |
| < 1.0 | :x: No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT report it publicly via GitHub Issues. Instead, disclose it privately by sending an email to **drleminhbui@gmail.com**.

We will investigate the report and respond with an evaluation within 48 hours. If the vulnerability is verified, we will coordinate a fix and release a patched version before disclosing the issue publicly.
