# Support Guidelines

Need help with LLM Walkie-Talkie? Here's how to get support:

## 1. Search the Documentation
Before opening a request, please review:
- The [README](README.md) for configuration setup and usage guidelines.
- The [walkie_details.txt](walkie_details.txt) architecture file for technical internals.

## 2. GitHub Issues
If you encounter bugs, parsing issues, or integration errors, please open a GitHub Issue describing:
- The model used (`groq/...`, `openrouter/...`, `nvidia/...`).
- Command parameters executed.
- Complete logs output (redacting any private credentials).

## 3. Direct Contact
For non-technical questions or private discussions, contact the maintainer at **drleminhbui@gmail.com**.
